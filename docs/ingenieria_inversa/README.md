# 🔍 Análisis de Ingeniería Inversa y Diagramación: GastroFlow

Este directorio contiene el análisis de arquitectura lógica y base de datos de GastroFlow obtenido por ingeniería inversa de su base de datos relacional y código fuente.

## 📌 Documentos Disponibles

* **[1. Modelo Entidad-Relación (MER)](MER.md):** Detalle técnico de las tablas, atributos, tipos de datos, llaves primarias/foráneas y relaciones de cardinalidad.
* **[2. Diagrama y Matriz de Casos de Uso](CASOS_DE_USO.md):** Identificación de actores del sistema y casos de uso con su flujo operacional.
* **[3. Diagrama de Actividad Completo](DIAGRAMA_ACTIVIDAD.md):** Flujo de actividades que representa la operación de punta a punta del restaurante.
* **[4. Diagramas de Flujo Algorítmicos](DIAGRAMAS_FLUJO.md):** Detalle lógico de algoritmos clave (cálculo de recetas, costeo de platos y resolución multi-tenant).
